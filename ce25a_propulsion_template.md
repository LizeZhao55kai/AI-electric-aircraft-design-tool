# 6 核心功能（系统级）

| 功能 ID | 功能名称 | 简要描述 |
|----------------|---------|---------|
| EPS-F-01 | 提供推进轴功率 | 将 800 V 高压直流电能转换为机械轴功率，驱动涵道风扇产生推力 |
| EPS-F-02 | 推力大小连续可调 | 接收飞控推力指令 (0–100%)，实时调节电机扭矩输出 |
| EPS-F-03 | 状态监控与上报 | 实时采集转速、扭矩、温度、电流、电压、故障字，按 CAN 总线周期性上报 |
| EPS-F-04 | 故障保护 | 过欠压保护、过温保护、过流保护、转子位置异常保护、CAN 通信中断保护 |
| EPS-F-05 | 双控制器协调换向 | 两个 100 kW 控制器并联驱动同一电机，通过 CAN 同步换向 |
| EPS-F-06 | 高压预充与放电 | 上电时预充 DC-Link 电容，下电后主动泄放至安全电压 |
| EPS-F-07 | BIT 自检 | 上电自检 (POST)、连续自检 (CBIT)、维护自检 (IBIT) |
| EPS-F-08 | 紧急切断 | 接收紧急停车指令后 50 ms 内切断高压、停止扭矩输出 |

# 8 失效模式（用于 FHA / FMEA / 共模 / 安全性）

| 失效模式 | 飞机级影响 | 危险等级 | 概率门限 |
|---------|------------|----------------|---------|
| 单台电驱动完全丧失 | 推力损失 12.5%（8 台冗余） | MAJ（较大） | ≤ 1×10⁻⁵/FH |
| 多台电驱动同时丧失（共模） | 推力损失显著，可能无法维持平飞 | HAZ（危险） | ≤ 1×10⁻⁷/FH |
| 全部电驱动丧失 | 推力完全丧失 | CAT（灾难性） | ≤ 1×10⁻⁹/FH |
| 推力误增大 (runaway) | 操纵困难、可能超速 | HAZ | ≤ 1×10⁻⁷/FH |
| 推力误减小或抖动 | 操纵性下降 | MAJ | ≤ 1×10⁻⁵/FH |
| 高压绝缘故障 / HVIL 失效 | 维护人员触电风险 | HAZ | ≤ 1×10⁻⁷/FH |
| CAN 总线双通道同时丧失 | 失去推力指令通道 | HAZ | ≤ 1×10⁻⁷/FH |
| 控制器过温 / IGBT 烧毁 | 单通道功率减半 | MAJ | ≤ 1×10⁻⁵/FH |
